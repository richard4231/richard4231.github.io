Automatisches Erfassen von Screenshots

Diese Anleitung erklärt, wie du Vorschaubilder für die Simulationen erzeugst.

  1. Installiere Node (falls noch nicht vorhanden).
  2. Erzeuge die Daten-Datei aus der Haupt-README: 
    node scripts/generate_data.js
    Dies erzeugt docs/data.json.
    3. 3. Installiere `puppeteer-core` (empfohlen, verwendet den lokal installierten Chrome): 
    npm install puppeteer-core --save-dev
  3. Erzeuge Screenshots (mit lokal installiertem Chrome): 
    CHROME_PATH="/Applications/Google Chrome.app/Contents/MacOS/Google Chrome" node scripts/capture-screenshots.js
    Die Bilder werden in docs/screenshots/ abgelegt. Falls du keine Umgebungsvariable setzt, versucht das Skript den Standard-Pfad für Chrome auf macOS zu verwenden.
  4. Lokale Vorschau: du kannst einfach einen einfachen HTTP-Server verwenden: 
    cd docs
python3 -m http.server 8000
# dann öffne http://localhost:8000/
  5. Deployment: wähle in den GitHub Pages Einstellungen den Ordner /docs als Quelle oder kopiere die Dateien ins Root.

Falls manche Seiten dynamisch Inhalte nachladen, kann es nötig sein, die Wartezeit im Skript scripts/capture-screenshots.js zu erhöhen.
